Comment tester ?
  • Utilisez a-z-e pour jouer un beat
  • Cliquez sur Record START pour commencer à enregistrer votre séquence
  • Cliquez sur Record STOP lorsque vous avez fini
  • Cliquez sur Replay pour réécouter votre séquence
  • Cliquez sur Try pour lancer le "jeu solo" : vous devez reproduire votre séquence. Un score vous sera attribué en fonction de vos performances.
